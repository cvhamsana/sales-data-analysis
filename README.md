# Sales Data Analysis

## Project Overview

This project analyzes retail sales data using Python to identify business trends and insights.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

* Calculated total sales.
* Analyzed sales by region.
* Identified top-selling products.
* Visualized sales distribution by category.

## Key Insights

* Compared sales across different regions.
* Found the highest revenue-generating products.
* Studied category-wise sales distribution.

## Files in this Repository

* `sales_analysis.ipynb` - Jupyter Notebook containing the analysis.
* `train.csv` - Dataset used for analysis.
